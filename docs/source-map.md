# Карта соответствия разделов и источников контента

*Легенда:* **P** — первичный источник (факты/данные), **S** — вторичный (аналитика/выводы).
*Роли:* SME — эксперт предметной области, PO — продакт-оунер домена, EA — архитектор предприятия, Sec — безопасность, DevOps — платформа.
